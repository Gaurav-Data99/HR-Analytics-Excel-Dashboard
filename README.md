#  HR Analytics Dashboard (Excel Project)

##  Project Overview

This project demonstrates a complete HR analytics dashboard built using Excel.

The goal is to analyze employee attrition, salary trends, and workforce distribution.

---

##  Key Features

* Data Cleaning & Preprocessing
* Interactive Dashboard using Pivot Tables
* KPI Metrics:

  * Total Employees
  * Attrition Count
  * Attrition Rate
  * Average Salary
  * Average Age
* Slicers for filtering (Department, Gender, Attrition)
* Charts:

  * Bar Chart (Attrition by Department)
  * Pie Chart (Attrition Distribution)

---

##  Tools Used

* Microsoft Excel / WPS Office
* Pivot Tables
* XLOOKUP / IF / TRIM functions
